# Contributing to tafsiri-mcp

Thank you for your interest in this project.

## How to Contribute

This repository is primarily maintained by [Gabriel Mahia](https://github.com/gabrielmahia)
as part of the [AI-KungFU East Africa](https://github.com/gabrielmahia/nairobi-stack)
coordination infrastructure.

### Bug Reports & Feature Requests

Please open a [GitHub Issue](https://github.com/gabrielmahia/tafsiri-mcp/issues)
to report bugs or request features. Do not submit unsolicited pull requests.

### Feedback

For strategic feedback, collaboration proposals, or integration discussions,
open an Issue with the `feedback` label or email contact@aikungfu.dev.

## Code of Conduct

Be respectful. Focus on the problem, not the person.
